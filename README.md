# 🚀 My Kubernetes Journey

Questo è il mio spazio di sperimentazione dedicato a **Kubernetes**. La repository nasce come un diario di bordo durante il mio tirocinio, con un focus particolare sul mondo dei **Digital Twin** e dell'**IIoT**.

## 🎯 Cosa troverai qui?
È tutto un **work in progress**. Man mano che studio e sperimento, aggiungo pezzi al puzzle. L'idea è quella di raccogliere:
* **Appunti teorici** per fissare i concetti che incontro lungo la strada.
* **Esperimenti pratici** (Labs) documentati, dai primi passi con Minikube a configurazioni più articolate.
* **Snippet e Manifest** pronti all'uso per non dover reinventare la ruota ogni volta.

## 🛠️ Il mio Setup attuale
Al momento sto lavorando in un ambiente isolato basato su **VirtualBox** con **Ubuntu Server**, usando **Minikube** come palestra per l'orchestrazione.
